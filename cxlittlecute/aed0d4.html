新时间日期API

之前经常使用的Data和Calendar等时间API其实并不怎么好用，麻烦而且存在线程安全问题，所以在JAVA8推出了新的API接口。

全新的API是不可变的线程安全的。

时间操作是我们写程序经常会用到的API，所以通过这篇文章进行一个总结。

本地时间和时间戳

  • LocalDate LocalTime LocalDateTime 它们的使用方式都一样 注意都是产生新的实例
public void test2(){
    //获取当前时间日期 now
    LocalDateTime ldt1 = LocalDateTime.now();
    System.out.println(ldt1);

    //指定时间日期 of
    LocalDateTime ldt2 = LocalDateTime.of(2020, 05, 17, 16, 24, 33);
    System.out.println(ldt2);

    //加 plus
    LocalDateTime ldt3 = ldt2.plusYears(2);
    System.out.println(ldt3);

    //减 minus
    LocalDateTime ldt4 = ldt2.minusMonths(3);
    System.out.println(ldt4);

    //获取指定的你年月日时分秒... get
    System.out.println(ldt2.getDayOfYear());
    System.out.println(ldt2.getHour());
    System.out.println(ldt2.getSecond());
    System.out.println(ldt2.getYear());
}
  • Instant:时间戳（以 Unix 元年 1970-01-01 00:00:00 到某个时间之间的毫秒值）
public void test3(){
    // 默认获取 UTC 时区 (UTC：世界协调时间)
    Instant ins1 = Instant.now();
    System.out.println(ins1);

    //带偏移量的时间日期 (如：UTC + 8)
    OffsetDateTime odt1 = ins1.atOffset(ZoneOffset.ofHours(8));
    System.out.println(odt1);

    //转换成对应的毫秒值
    long milli1 = ins1.toEpochMilli();
    System.out.println(milli1);

    //构建时间戳 加60秒
    Instant ins2 = Instant.ofEpochSecond(60);
    System.out.println(ins2);
}
  • Duration：计算两个时间的间隔
  • Period：计算两个日期之间的间隔
public void test4() throws InterruptedException {
    Instant now1 = Instant.now();
    Thread.sleep(1000);
    Instant now2 = Instant.now();

    Duration between = Duration.between(now1, now2);
    // 毫秒这里是toXXXX()，秒是getXXX（）
    System.out.println(between.toMillis());

    System.out.println("------------------------------");

    LocalTime lt1 = LocalTime.now();
    Thread.sleep(3000);
    LocalTime lt2 = LocalTime.now();
    Duration between2 = Duration.between(lt1, lt2);
    System.out.println(between.toMillis());

    System.out.println("-------------注意：上面都是获取时间的间隔-------------");
    System.out.println("-------------注意：下面都是获取日期的间隔-------------");

    LocalDate ld1 = LocalDate.of(2019, 10, 20);
    LocalDate ld2 = LocalDate.now();

    Period period = Period.between(ld2, ld2);
    System.out.println(period); // 输出的是ISO标准的格式
    // 可以进一步获取详细信息
    System.out.println(period.getYears());
    System.out.println(period.getMonths());
}

时间校正器

  • TemporalAdjuster：时间校正器。有时我们需要调整日期为想要的时间。这个类是一个接口。
  • TemporalAdjusters：该类提供了很多静态方法进行实现。
@FunctionalInterface
public interface TemporalAdjuster {
    Temporal adjustInto(Temporal var1);
}

这里提一下Temporal接口，大家可以看一下LocalDate这些本地时间操作的源码，它是实现了这个Temporal这个接口的，所以Temporal可以根据实际需求表示为日期、时间或两者组合。

public void test5(){
    LocalDateTime now = LocalDateTime.now();
    // 将月中的天数指定为想要的天数
    // 改为本月十号
    LocalDateTime localDateTime = now.withDayOfMonth(10);
    // with（）传递一个时间校正器
    // 下一个周五是什么时候
    LocalDateTime with = now.with(TemporalAdjusters.next(DayOfWeek.FRIDAY));
    System.out.println(with);
}

我们可以根据源码看出TemporalAdjuster是一个函数式接口，那么我们可以通过自定义时间来判断多久是下一个工作日：

LocalDateTime with1 = now2.with((t) -> {
    LocalDateTime ldt4 = (LocalDateTime) t;
    DayOfWeek dow1 = ldt4.getDayOfWeek();
    if (dow1.equals(DayOfWeek.FRIDAY)) {
        return ldt4.plusDays(3);
    } else if (dow1.equals(DayOfWeek.SATURDAY)) {
        return ldt4.plusDays(2);
    } else {
        return ldt4.plusDays(1);
    }
});
System.out.println(with1);

时间格式化

-DateTimeFormatter：格式化时间 / 日期

public void test6(){
    // DateTimeFormatter后能选择很多种格式化的方式
    DateTimeFormatter dtf1 = DateTimeFormatter.ISO_DATE_TIME;
    LocalDateTime ldt1 = LocalDateTime.now();
    String str1 = ldt1.format(dtf1);
    System.out.println(str1);

    //自定义格式化 ofPattern
    DateTimeFormatter dtf2 = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
    LocalDateTime ldt2 = LocalDateTime.now();
    String str2 = ldt2.format(dtf2);
    System.out.println(str2);

    //解析回去 
    LocalDateTime newDate = ldt1.parse(str1, dtf1);
    System.out.println(newDate);
}

时区操作

  • ZonedDate
  • ZonedTime
  • ZonedDateTime
    public void test7(){
    //查看支持的所有时区
    Set<String> set = ZoneId.getAvailableZoneIds();
    set.forEach(System.out::println);

    //指定时区 构建时间
    LocalDateTime ldt1 = LocalDateTime.now(ZoneId.of("Europe/Tallinn"));
    System.out.println(ldt1);

    //在已构建好的日期时间上指定时区
    LocalDateTime ldt2 = LocalDateTime.now(ZoneId.of("Europe/Tallinn"));
    ZonedDateTime zdt1 = ldt2.atZone(ZoneId.of("Europe/Tallinn"));
    System.out.println(ldt2);
    System.out.println(zdt1);
}
