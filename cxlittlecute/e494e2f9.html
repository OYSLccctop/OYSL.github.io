DFS BFS

上一篇博客记录了自己初次学习图遍历的代码实现方法，有朋友评论讲的过于细致化，今天刚好重新参考看了一篇博客的实现，相比来说确实要简单明了很多，于是再次记录关于DFS和BFS的相关内容。

如果上一篇博客没有太明白的朋友，可以参考看一看这一篇。

上一篇博客我们说了图的实现方式主要是邻接矩阵和邻接表，那么今天我们就用邻接表再次分析图的DFS和BFS过程。

图的实现

  • 使用一个大小为V的数组 List[] head（通图adj[]），把索引看做是顶点
  • 每个索引处head[v]存储了一个数组或队列，该队列中存储的是所有与该顶点相邻的其他顶点
/**
 * @ClassName Graph2
 * @Description TODO
 * @Auther OY
 * @Date 2020/7/25 14:12
 * @Version 1.0
 **/
public class Graph2 {
    // 记录顶点的个数
    private final int V;
    // 保存头结点
    List<Integer>[] head;
    // 记录边的数量
    private int E;
    Graph2(int V) {
        this.V = V;
        this.E = 0;
        this.head = new ArrayList[V];
        // 初始化邻接表中的队列
        for(int i = 0;i < V ;i++){
            head[i] = new ArrayList<Integer>();
        }
    }
    // 获取图中顶点的数量
    public int V() {
        return V;
    }

    // 获取图中边的数量
    public int E() {
        return E;
    }

    // 向图中添加一条边
    public void addEdge(int v, int w) {
        head[v].add(w);
        head[w].add(v);
        E++;
    }
    //获取和顶点v相邻的所有顶点
    public List<Integer> adj(int v){
        return head[v];
    }

}

DFS的实现

所谓的深度优先搜索，指的是在搜索时，如果遇到一个结点既有子结点，又有兄弟结点，那么先找子结点，然后找兄弟结点

我们可以根据上图清楚的知道DFS的一个执行顺序。
很明显，在由于边是没有方向的，所以，如果 4和5顶点相连，那么4会出现在5的相邻链表中，5也会出现在4的相邻链表中，那么为了不对顶点进行重复搜索，应该要有相应的标记来表示当前顶点有没有搜索过，可以使用一个布尔类型的数组 boolean[V] marked,索引代表顶点，值代表当前顶点是否已经搜索，如果已经搜索，标记为true，
如果没有搜索，标记为false；

主要有两点需要注意：

  1. 如果已经访问过的会直接跳过。
  2. 由于是DFS是深度优先，所以我们每次遍历都会以邻接结点为下一次遍历的头。（特点）
/**
 * @ClassName DepthFirstSearch
 * @Description TODO
 * @Auther OY
 * @Date 2020/7/25 17:42
 * @Version 1.0
 **/
public class DepthFirstSearch {
    private boolean[] marked;
    private int count;
    DepthFirstSearch(Graph2 G,int s){
        this.marked  = new boolean[G.V()];
        this.count = 0;
        dfs(G,s);
    }
    // 使用深度优先搜索找出G图中v顶点的所有相通顶点
    private void dfs(Graph2 G, int v){
        System.out.print(v+"->");
        // 把v顶点标识为以搜索
        marked[v] = true;
        for (Integer w : G.adj(v)) {
            // 判断有没有被搜索过，如果没有被搜索过则递归调用dfs进行深度搜索
            if (!marked[w]){
                dfs(G, w);
            }
        }
        count++;
    }
    // 判断w顶点与s顶点是否相通
    public boolean marked(int w){
        return marked[w];
    }
    // 获取与顶点s相通的所有顶点的总数
    public int count(){
        return this.count;
    }
}

BFS实现

而BFS刚好相反，指的是在搜索时，如果遇到一个结点既有子结点，又有兄弟结点，那么先找兄弟结点，然后找子结点。

如果我们现在以0为顶点，它会先去找0这个顶点的邻接表，也就是有6、2、1、5这几个相邻的顶点，然后我们分别以这几个相邻的点为顶点去找它们的邻接表。（类似于二叉树的层序遍历）

/**
 * @ClassName BreadthFirstSearch
 * @Description TODO
 * @Auther OY
 * @Date 2020/7/25 21:02
 * @Version 1.0
 **/
public class BreadthFirstSearch {
    // 索引代表顶点，值表示当前顶点是否已经被搜索
    private boolean[] marked;
    // 记录有多少个顶点与s顶点相通
    private int count;
    // 用来存储待搜索邻接表的点
    private Queue<Integer> waitSearch;

    // 构造广度优先搜索对象，使用广度优先搜索找出G图中s顶点的所有相邻顶点
    BreadthFirstSearch(Graph2 G, int s) {
        this.marked = new boolean[G.V()];
        this.count = 0;
        this.waitSearch = new Queue<Integer>();
        bfs(G, s);
    }

    // 使用广度优先搜索找出G图中v顶点的所有相邻顶点
    private void bfs(Graph2 G, int v) {
        // 把当前顶点标记为已搜索
        marked[v] = true;
        // 让顶点v进入队列，待搜索
        waitSearch.enqueue(v);
        // 通过循环，如果队列不为空，则从队列中，则从队列汇总弹出一个待搜索的顶点，进行搜索
        while (!waitSearch.isEmpty()) {
            // 弹出待搜索的顶点
            try {
                Integer wait = waitSearch.dequeue();
                // 遍历wait顶点的邻接表
                for (Integer w : G.adj(wait)) {
                    // 判断是否被搜索
                    if (!marked[w]){
                        bfs(G, w);
                    }
                }

                /* 第二种方式：
                for (Integer w : G.adj(wait)) {
                    if (!marked[w]){
                        System.out.print(w+"->");
                        waitSearch.enqueue(w);
                        marked[w] = true;
                    }
                }*/
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
        // 想通的结点+1
        count++;
    }

    // 判断w顶点与s顶点是否相通
    public boolean marked(int w) {
        return marked[w];
    }

    // 获取与顶点s相通的所有顶点的总数
    public int count() {
        return count;
    }

}

注意：BFS会把兄弟结点访问完毕了，也就是队列中的结点，然后再以队列中的每个结点为头找他们的子结点。
